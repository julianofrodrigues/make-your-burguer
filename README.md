## 📁 Project Access
To try it is very simple:<br>
-Clone the repository
```bash
$ git clone 
```
-Download the libraries
```bash
$ yarn install
```
Or
```bash
$ npm install
```
-Start the Project
```bash
$ yarn dev
```
Or
```bash
$ npm run dev
```
